== Description ==
AI Text Enhancer is a powerful tool that improves your writing by fixing grammar, enriching vocabulary, and optimizing your style. With advanced AI technology, it transforms your text into a polished, engaging piece.

The plugin relies on a 3rd party service called [WordAI](https://wordai.com) ([Terms of Service](https://wordai.com/terms-of-service)|[Privacy Policy](https://wordai.com/privacy-policy)). 
You will need a WordAI API key to use the AI text improvement functionality. 

In the PRO version you can use ChatGPT-3.5 as AI provider.


== Installation ==
1. Upload the plugin files to the /wp-content/plugins/ai-text-enhancer directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the Plugins screen in WordPress
3. Select and setup the AI api connection in the plugin settings


== Frequently Asked Questions ==

= What AI apis are supported? =

Currently it is WordAI. You can use ChatGPT in the PRO version.
